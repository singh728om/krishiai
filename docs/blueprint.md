# **App Name**: KrishiAI

## Core Features:

- Farmer Dashboard & Navigation: Centralized hub for farmers with key performance metrics, real-time alerts, farm overview (health map), and access to all application features through intuitive sidebar navigation. Includes multi-lingual display for content and metrics.
- Crop Disease Detection Tool: AI-powered tool allowing farmers to upload crop photos for instant disease identification, confidence scores, severity assessment, and vernacular treatment recommendations. Includes a history of past scans.
- Yield Prediction & Advisory Tool: Farmers input farm parameters (crop type, acreage, soil, irrigation, sowing date) to receive AI-driven yield forecasts, historical yield comparisons, and personalized advisories for yield improvement.
- Carbon Credit Management: Dashboard to track CO₂ sequestered, manage earned, sold, and pending carbon credits. Features a verification timeline, live price ticker, marketplace information, and payout history.
- Market Intelligence & Selling Advice: Provides real-time commodity prices, compares with Minimum Support Price (MSP), predicts optimal selling times using an AI tool, lists nearby mandis with directions, and displays eNAM integration status.
- FPO / Cooperative Management: B2B portal enabling cooperatives to manage farmer members, view aggregate analytics (total land, avg yield, carbon credits), and send bulk advisories to their farmer base.
- User Profile & Settings: Allows farmers to manage farm details, select preferred vernacular language for the application, configure notification preferences, and view their subscription plan.

## Style Guidelines:

- Primary Color: Deep, earthy green (#16a34a) for a strong connection to agriculture, growth, and nature. This color will be used for interactive elements and key branding.
- Background Color: A very light, desaturated green-gray (#F6FCF7) as the primary background, offering a clean, modern, and breathable canvas for content, aligning with a light theme.
- Accent Color: A vibrant amber (#d97706) to highlight critical alerts, calls to action, important data points, and to add a touch of warmth and vibrancy to the interface.
- Neutral Colors: Clean whites and various shades of gray will be utilized for text, borders, shadows, and secondary elements to maintain clarity, professionalism, and a modern aesthetic.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, slightly tech-inspired, and impactful appearance, suitable for headlines and titles. Body font: 'Inter' (sans-serif) for optimal readability and a neutral, professional feel in paragraphs and data displays.
- Utilize 'Lucide React' for a comprehensive set of clean, consistent, and scalable vector icons that seamlessly integrate with the modern UI aesthetic and dashboard functionality.
- Employ a responsive, mobile-first design with Tailwind CSS for styling. The layout features a persistent top header for critical info and a collapsible sidebar navigation. Content is organized in distinct cards with subtle shadows and hover effects for enhanced interaction.
- Incorporate subtle loading animations like spinners and skeleton loaders for AI-powered features (disease detection, yield prediction). Utilize smooth transitions and animated numerical displays for dynamic metrics like carbon credits and yield forecasts to provide engaging user feedback.
